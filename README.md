# CodeAlpha Social Media Platform

## Project Overview

This project was developed as part of the CodeAlpha Full Stack Development Internship.

The application allows users to register, log in, create posts, like posts, comment on posts, and interact with other users through a simple follow system.

---

## Features

### User Authentication
- User Registration
- User Login
- JWT Authentication

### Social Features
- Create Posts
- View Posts
- Like Posts
- Add Comments
- Follow Users
- User Profiles

### Database
- Users Collection
- Posts Collection
- Comments Collection
- Followers Management

## Installation

### Clone Repository

git clone <repository-url>

### Install Dependencies

cd backend

npm install

### Run Backend

npm start

### Run Frontend

Open frontend/index.html using Live Server.

---

## API Endpoints

### Authentication

POST /api/auth/register

POST /api/auth/login

### Posts

GET /api/posts

POST /api/posts

POST /api/posts/:id/like

POST /api/posts/:id/comment

### Follow System

POST /api/users/:id/follow
